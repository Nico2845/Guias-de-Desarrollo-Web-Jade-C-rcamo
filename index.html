Descripción del Curso

Este curso introduce los fundamentos esenciales del desarrollo web: HTML, CSS y JavaScript. A través de estos lenguajes, los estudiantes aprenderán a construir sitios web funcionales, visualmente atractivos y con contenido dinámico.

Contenidos

UNIDAD I: INTRODUCCIÓN AL HTML

  • Elementos HTML.
  • Semántica del HTML.
  • Atributos HTML.

UNIDAD II: INTRODUCCIÓN A LAS HOJAS DE ESTILO EN CASCADA (CSS)

  • Selectores CSS.
  • Propiedades CSS.
  • Modelo de caja.

UNIDAD III: MAQUETACIÓN DE SITIOS WEB

  • Posicionamiento y visualización.
  • Diseño de layout básico.
  • Diseño responsivo y media queries.
  • Introducción a Flexbox.
  • CSS Grid.

UNIDAD IV: FRAMEWORKS DE DISEÑO WEB RESPONSIVO

  • Malla de Bootstrap.
  • Utilidades para Bootstrap.
  • Elementos Bootstrap.

UNIDAD V: INTRODUCCIÓN A JAVASCRIPT

  • Introducción a Javascript y su sintaxis.
  • Estructuras de control.
  • Arreglos y objetos.
  • Funciones y eventos.
  • Programación Orientada a Objetos en Javascript.

UNIDAD VI: EL MODELO DE OBJETOS DEL DOCUMENTO (DOM) Y JAVASCRIPT ASÍNCRONO

  • Manipulación del Document Object Model (DOM).
  • Notación de Objetos de Javascript (JSON).
  • Almacenamiento local: localStorage y sessionStorage.
  • Programación asíncrona en Javascript.
  • AJAX.
  • Consumo de APIs.
  • Almacenamiento de datos en Firebase.
